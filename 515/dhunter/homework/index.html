Regarding Sweave and .Rnw files

  • To use Sweave on an existing .Rnw file: In R, type Sweave("path and filename here"). This will create a .tex file in your current R directory that can be compiled using pdflatex. For example, try the following steps:
    • Copy sol04.Rnw to some directory on your computer.
    • Start R from that directory or, alternatively, start R and then use the setwd() function to set your working directory to be that directory.
    • In R, Type Sweave("sol04.Rnw"). This creates two new files in your working direcory: A LaTeX file called sol04.tex and a figure pdf file called sol04-006.pdf.
    • The file sol04.tex may now be compiled using pdflatex or whatever you use to turn LaTeX files into pdf. The sol04-006.pdf file is important; it will be used in the LaTeX compilation process.
    • After compiling the .tex file, you will have created sol04.pdf.
  • To create an Sweave file yourself: Any LaTeX file can be renamed with the .Rnw extension and thus it becomes an Sweave file. For example, you can simply save a copy of hw04.tex as hw04.Rnw and then use the Sweave compilation process described above.
  • To learn more about Sweave: The Sweave homepage is maintained by the Sweave developer, Friedrich Leisch. From this page you can link to a manual and FAQ.
